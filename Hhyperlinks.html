hyperlinks and a tag

A hyperlink is a clickable link that takes the user to another website, article or email address or location.
(a) — anchor tags are used to create links.
(a href="Url"> some text (/a) target="_self" "_blank"

- href: is an attribute, it is hypertext reference, it tells the browser where to go.

- target: this is the target attribute, it tells where the browser where it should open the link, _self for it to open in the same tab or the _blank in another tab.

- title attribute: it shows text when a cursor or mouse hovers over the link.

You can also link other files, just put the file name.
