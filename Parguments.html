Arguments

Default Arguments
A default value for certain parameters, default is used when that argument is excluded. makes your functions more flexible, reduces # of arguments.
def net_price(list_price, discount=0, tax=0.05):
return list_price * (1 - discount) * (1 + tax)

Positional Arguments
Values passed into a function based on their order / position
def greet(First, Last):
print(f"hello {First} {Last}")

greet("Krapopolis", "Ramius")

(First) and (Last) are in position
(First) and "Ramius" are in position

Keyword arguments

An argument preceded by an identifier, helps with readability
(order doesn't matter)

def hello(greeting, title, First, Last):
print(f"greetings {title} {First} {Last}")


hello("hello", title="Mr.", last="Ramius", First="Krapopolis")
Output: hello Mr. Krapopolis Ramius
