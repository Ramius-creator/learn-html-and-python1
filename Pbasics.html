Variables

Variables behave as if they are the value they contain.

Player = "Ramius" ← Value assigned to the variable

How to use f-strings

"f" means format.

Print(f"Hello {Player}")
Output: Hello Ramius

Variable names can be almost anything, such as:

Player
Name
Place
Game

Do not put quotation marks around a variable name when you want its value.

Correct:
Print(Player)
Output: Ramius

Incorrect:
Print("Player")
Output: Player

The curly braces { } inside an f-string tell Python to insert the value of a variable.

Other examples of variables

Place = "Bago Gallera"
Print(f"You are from {Place}")
Output: You are from Bago Gallera

Game = "Roblox"
Print(f"You like {Game}")
Output: You like Roblox ```
