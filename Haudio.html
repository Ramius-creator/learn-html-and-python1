audio

(audio controls loop autoplay muted) (source src="song.mp3" type="audio/mpeg") - Audio tag: creates an audio player on the webpage, and creates the necessary audio.

- controls attribute: it adds the play button, pause button, volume control, timeline bar; without this, the user can’t interact with the audio.

- source tag: tells the browser where the audio file is located.

- type attribute: you can tell the browser what type of audio file you’re using.

- muted: it mutes the audio, better partnered with loop 🙂

- loop: this will repeat the audio, repeated; better to mute it tho.

- autoplay attribute: it makes the audio start playing automatically — is when the browser / page loads.
