LISTS

```text it’s called lists for a reason 😊

3 types of lists:

- unordered
- ordered
- description

(li) means list item, used for making lists.
(ul) means unordered list, used to make unordered lists.
(ol) means ordered list, used to make ordered lists.
(dl) means description list, used for terms and their descriptions.
(dt) means description term, the term being described.
(dd) means description details (or description definition), the description of the term.

Unordered List

html

(ul)
(li) milk (/li)
(li) eggs (/li)
(li) bread (/li)
(li) coffee supplies (/li)
(li) creamer (/li)
(li) sugar (/li)
(/ul)

Output:

- milk
- eggs
- bread
- coffee supplies
- creamer
- sugar

another list can be made inside a list item, it will usually show a hollow bullet point.

Ordered List

html

(ol)
(li) eat breakfast (/li)
(li) go to class (/li)
(li) walk dog (/li)
(li) work (/li)
(/ol)

Output:

1. eat breakfast
2. go to class
3. walk dog
4. work

Description List

html

(dl)
(dt) HTML (/dt)
(dd) The standard language used to create web pages. (/dd)

(dt) CSS (/dt)
(dd) Used to style and format web pages. (/dd)
(/dl)

Notes:

- (dl) creates a description list.
- (dt) contains the term being described.
- (dd) contains the description or definition of that term.
- Description lists are useful for glossaries, definitions, FAQs, and key-value style information. ```
