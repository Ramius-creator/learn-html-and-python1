Favicon (Favorite Icon)

Favicons are the small images shown on browser tabs, bookmarks, and browser history entries.

32 × 32 pixels
16 × 16 pixels
are common recommended sizes.

html

(head)
(title) My Website (/title)
(link rel="icon" href="favicon.png")
(/head)

Notes:

- A favicon is the small icon displayed on a browser tab next to the page title.
- You can create a favicon yourself or use an image and convert it to a .ico file if desired.
- Common file formats are .ico, .png, .jpg, and .svg, though .ico and .png are the most common.
- Add the favicon code inside the (head) section of your HTML document.
- (link): creates a connection between your HTML page and an external resource.
- rel="icon": tells the browser that the linked file is a favicon.
- href="favicon.png": tells the browser where to find the image file. If no path is specified, it looks in the same folder as the HTML file.

Output:

[🌐] My Website

(The favicon appears on the browser tab next to the page title.)
