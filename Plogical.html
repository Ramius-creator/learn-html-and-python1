Logical operators

Logical operators or → at least one condition must be true
and → both conditions must be true
not → inverts the condition (true → false, false→true)

OR logical operator
age = 10
if age < 13 or age > 19:
Print("not a teenager")
Output: not a teenager


AND logical operator
username = "Ramius"
password = "100"
if username == "Ramius" and password == "100":
Print("logged in")
Output: logged in


NOT logical operator
Logged_in = False
if not Logged_in:
print("please log in")
Output: please log in
