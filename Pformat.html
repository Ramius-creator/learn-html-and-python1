format specifiers

Format a value based on what Flags are inserted
Price = 3.14159
Price2 = -987.65

Print(f"Price 1 is {price:.2f}")
Output: 3.14 Price 1 is 3.1


(:)allows how the decimal
(2)tells the program how many
(f)this is the place where decimals
(2)can be changed
(.) is just necessary to put there
note: look at the example to understand

Other examples of Format specifiers

money = 1000000
Print(f"money:,3")
(between the "money" and the last " " " is where you put the format specifiers)

(if you put a comma between the (:) and (3), as shown it will put the comma in the output in a designated place.)

Output: 1,000,000

another example
Score = 0.76
Percentage % being used
Print(f"your score is {score:.0%}")
Output: your score is 76%
just like the first example

another example name = Ramius
Print(f"{name:<10}") → can be used for right justify
Print(f"{name:>10}") → can be used for left justify
Print(f"{name:^10}") → can be used for what this is
Output: Ramius
