Tables

html

(table border="1")
(tr)
(th) Name (/th)
(th) age (/th)
(/tr)
(tr)
(td) Ramius (/td)
(td) 16 (/td)
(/tr)
(/table)

Output:

Name | age
Ramius | 16

- table tag: this tag creates the table.
- table row: this creates a table row. Make another (tr) to create another row.
- table header: it becomes the column title.
- table data: text inside these tags appears below the column title. ```
