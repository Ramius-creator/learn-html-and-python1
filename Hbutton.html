buttons

```text Buttons

Buttons allow users to interact with web pages.

html

(a href="URL")
(button title="This is a button") click me (/button)
(/a)

(button onclick="alert('Hello')") click to show message (/button)

(button disabled) This means it's disabled (/button)

Notes:

- Wrapping a button inside an (a) tag makes it a hyperlink. You can link to other pages, files, or websites.
- Button tag: used to create buttons. Place text between the opening and closing tags to display words on the button.
- Title attribute: when the cursor hovers over the button, the specified text appears as a tooltip.
- Onclick: when the button is clicked, it runs JavaScript. In this example, it displays a popup message.
- Disabled: makes the button inactive so it cannot be clicked. ```
