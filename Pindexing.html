indexing

Accessing elements of a sequence using [] (Indexing Operator)
[start: end : step]


Numbers = 1 2 3 4 5 6 7 8
(python starts counting at 0)

Print(Numbers[2]) → put any number from 1-7
(if you put [2] the output will be 3)
(if you put [4] the output will be 5)


you can print the whole set by doing this [: : ]
and you can do this at any number or letter
output: 1 2 3 4 5


(just put nothing after the colon) if you want it to print the end of the string at starting number
if you want to start at the end of the string, just put (negative) and then the number you want in the string


use this [::2] to print even two numbers/letters: 2 4 6 8
use this [::3] to print every third number/letter: 3 5 7
and so forth
