Forms

Forms allow users to enter and send information.

html

(form action="submit.php" method="POST")

(label) Name: (/label)
(input type="text" placeholder="Enter your name")
(br)

(label) Email: (/label)
(input type="email" placeholder="your.email@example.com")
(br)(br)

(label) Password: (/label)
(input type="password" placeholder="Password")
(br)(br)

(input type="submit" value="Register")
(/form)

Notes:

- (form) creates a container for all form elements. When submitted, the data is sent to the file specified in the action attribute.
- action: specifies where the form data is sent when the form is submitted.
- method: determines how the data is sent. Common values are "GET" and "POST".
- (label) displays text such as "Name:" next to an input field.
- (input type="text"): creates a text box for entering regular text, such as a name.
- placeholder: displays hint text inside the input field, such as "Enter your name".
- Very important! To actually send the user's input, you should also add a name attribute. Without it, the field's data is usually not included when the form is submitted.
- (br): creates a line break or spacing between elements.
- (input type="email"): creates an email input field and can check if the entered text looks like a valid email address.
- (input type="password"): creates a password field that hides the characters being typed.
- (input type="submit"): creates a submit button that gathers the form data and sends it to action="submit.php".

Output:

Name: [ Enter your name ]
Email: [ your.email@example.com ]
Password: [ •••••••• ]
[ Register ]
